Reading List

Invest uses a lot of standard technologies and practises. We recommend you familiarise yourself with the following before starting:

  • GraphQL is a query language which provides the API between the UI and the server.

If you want to work on the server-side:

  • Java 8 is the language used for the server.
  • Spring Boot, is the underlying framework and specifically we use Webflux and Spring Data.
  • GraphQL SQPR which is an annotation based approach to creation of GraphQL services in Java.
  • Project Reactor and in particular the use of Flux and Mono.
  • Maven for server-side dependency management and building plugins.

If you want to work on the UI, we'd suggest reading about:

  • Yarn is used for dependency packaging our UI.
  • React is a library for building user interfaces for the Web.
  • Typescript is an Javascript like language adding classes and interfaces.
  • React Semantic UI is a library of a styled UI components for React.
  • Apollo and React-Apollo is a Javascript client for working with GraphQL.

On the UI side you don't have to use these but we do!

Developing

In order to develop, you require the following pre-requisites to be installed:

  • Java 8
  • Maven
  • Yarn, though it is worth also installing NodeJS and NPM
  • Lerna
  • Git

You are likely to want to access existing data in a supported database, as such you should also install and setup the appropriate database for example:

  • Mongo 3.4.4 or above
  • Elasticsearch 5

Of course you can use whatever deployment approach you want such as local installation, Docker constainers, or cloud services.

You'll probably want to use an IDE, we recommend:

  • Visual Studio Code or Atom for the UI (with appropriate Typescript, etc plugins)
  • Intellij IDEA or Eclipse for server development. We'd recommend the Eclipse based Spring Tool Suite instead of plain Eclipse since it has the additional components needed.
  • A Git GUI such as Sourcetree, Github Desktop, etc.
